# Emergency Health Card - VitalLink

An AI-powered Emergency Health Management Interface for registering patient details, generating vital statistics ID cards, and managing emergency health information.

## Features

- **Health Card Generation** - Register new patient details and generate vital statistics ID cards instantly
- **Emergency Management Hub** - Central dashboard for managing health-related modules
- **AI API Integration** - Configurable AI API settings for enhanced functionality
- **Database Management** - Built-in database settings configuration
- **Real-time Clock** - Live time display in the header
- **Modern UI** - Glass-morphism design with animated background and responsive interface

## Project Structure

```
app/
├── index.html      # Main HTML structure and views
├── app.js          # Core application logic and navigation
├── app.css         # Styling and animations

```

## Getting Started

1. Open `index.html` in a modern web browser
2. Use the navigation menu to access different modules
3. Configure AI API and Database settings via the settings buttons in the top-right corner

## Technologies

## Key Modules

- **Dashboard** - Main hub for module selection
